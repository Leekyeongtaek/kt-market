# KT-MARKET 포트폴리오
## 목차
- 프로젝트 상세
- 요구사항 분석
- 테이블 ERD
- 엔티티 ERD
- 핵심 기능과 코드
- 요구사항과 테스트 코드
</br>
### 프로젝트 상세
- Java : 17
- SpringBoot : 3.2.4
- MySql : 8.0.32
- QueryDSL : 5.0.0
</br>
### 요구사항 분석
- 기능 목록
  - 회원 기능
    - 회원 등록
    - 회원 조회
  - 상품 기능
    - 상품 등록
    - 상품 수정
    - 상품 조회
    - 상품 목록 조회
  - 영수증 기능
    - 영수증 발급
    - 영수증 조회
    - 영수증 목록 조회
</br>
